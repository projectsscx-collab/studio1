# **App Name**: CoverMe Forms

## Core Features:

- Personal Details Form: Collects user's personal information using standard input fields with appropriate labels.
- Contact Details Form: Collects user's contact information including address, phone number, and email using standard input fields.
- Demographic Information Form: Collects user's demographic information using multiple choice and single entry form fields.
- Real-Time Input Validation: Validates input fields in real-time to ensure data accuracy and completeness.
- Submission Confirmation: Displays a confirmation message after successful submission of the form.
- Form Generation Tool: Use AI to assist in deciding when to add new fields or form options to make data entry faster. This works like an assistant tool within the form that reduces unnecesary data entries.

## Style Guidelines:

- Primary color: Red (#B71C1C) from the MAPFRE logo and base. Use for main headers and key CTAs.
- Secondary color: White (#FFFFFF) for backgrounds to ensure a clean and readable interface.
- Accent color: Light green (#A5D6A7) for secondary CTAs and success states, providing a subtle contrast.
- Body and headline font: 'Roboto', a versatile sans-serif font ensuring readability and a modern appearance.
- Use a clear, step-by-step layout to guide the user through the form-filling process.
- Subtle transitions between form steps to provide a smooth user experience.
